## AI Integration

This setup has Claude Code and multiple AI models wired directly into the desktop вЂ” no browser tab needed.

- **Claude Code** вЂ” `Super + Alt + C` opens a Kitty terminal with Claude Code ready
- **AI Sidebar** вЂ” `Super + A` opens a sidebar with Claude, Gemini, Mistral and DeepSeek support
- **Local LiteLLM proxy** вЂ” running on `localhost:4099`, OpenAI-compatible API
- **Quick query** вЂ” select any text + `Super + Shift + Alt + right-click` в†’ answer as desktop notification
